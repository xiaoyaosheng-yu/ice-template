# ice-stark-layout

## 使用

- 安装依赖: `npm install`
- 启动调试服务: `npm start`
- 构建 dist: `npm run build`

## 目录结构

- 应用配置: `src/app.tsx`
- 路由配置: `src/routes.ts`
- 布局文件: `src/layouts`
- 通用组件: `src/components`
- 页面文件: `src/pages`

## 注意事项
1、本项目的react子应用对应的端口为3334(文件地址：ice-layout/src/app.tsx)，如有需要，请修改为你自己的地址
2、本项目的vue子应用对应的端口为8080(文件地址：ice-layout/src/app.tsx)，如有需要，请修改为你自己的地址
